New Feature: Timeframe for Merged Cancelled/Irregular Lessons

Normal Merged Lessons
Double/triple lessons show timeframe as usual (e.g., "08:00–09:30")
NEW: Cancelled Merged Lessons
Now show timeframe with crossed-out styling (e.g., "10:00–11:30")
NEW: Irregular Merged Lessons
Now show timeframe with normal styling (e.g., "13:00–14:30")
Single Cancelled/Irregular Lessons
Unchanged - timeframe remains hidden to save space
Key Detection: Merged Lessons

Lessons are detected as "merged" when they contain the merge separator " | " in either the subject name or lesson text. This indicates double or triple lessons that have been combined.
